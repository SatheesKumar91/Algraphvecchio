per clonazione:

git clone http://github.com/SatheesKumar91/Algraph.git

per aggiungere tutti i file alla repository:

git add *
git commit -m 'commit messaggio'
git push

per aggiungere un solo file alla repository:

git add PathFile.Extenzione

per aggiornare prima di iniziare a lavorare:

git pull 

per vedere lo stato:

git status
